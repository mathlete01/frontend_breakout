About

QWERTYBall is the realization of an idea I had years ago that it would be fun to create a twist on the classic game of Breakout by transposing my keyboard onto my screen so that you had to bang on the keys to keep the ball in play.

For whatever reason, the idea stuck with me, so when I decided to get back into programming and needed a project to work on, I decided to make this game for real.

I employed the Canvas API to dynamically draw the keyboard on the user’s screen, and I used the Web Audio API to add original sound effects to make the game feel more responsive

I hope you enjoy it. Feel free to get in touch if you have any feedback!

-Matty
